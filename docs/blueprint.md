# **App Name**: DEVELOP Real Estate Platform

## Core Features:

- Property Search Hub: Enable users to search for properties based on type (Apartments, Villas, Penthouses, Townhouses, Plots) and purpose (Buy, Rent, Commercial).
- Agent Directory: A searchable directory where users can find agents based on their specialization (neighborhood) or spoken language.
- Interactive Map Integration: Integrate a map interface showing real-time price heatmaps of Dubai, allowing users to search for properties visually.
- AI-Powered Property Recommendations: Utilize generative AI to provide personalized property recommendations based on user preferences and search history.
- VIP/Dark Mode Portal: Implement a password-protected 'Off-Market' section for high-net-worth investors with exclusive VIP listings.
- Trust Signals Display: Showcase statistics to build trust (Years of Experience, Number of Transactions, Total Value).

## Style Guidelines:

- Primary color: Metallic gold (#B3A369) for luxury and exclusivity.
- Background color: Deep charcoal (#1A1A1A) for an ultra-premium and sophisticated look.
- Accent color: Slate grey (#708090) for a subtle contrast.
- Body and headline font: 'Inter' for a modern, corporate feel. Note: currently only Google Fonts are supported.
- Use minimalist, geometric icons representing property types and features.
- Incorporate a high-impact video hero section showcasing Dubai skylines on the homepage.
- Subtle animations and transitions to enhance the user experience and create a dynamic feel.
